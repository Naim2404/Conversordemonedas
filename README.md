# Conversordemonedas
Es un conversor de monedas sencillo utilizando una API.

1. Herramientas utilizadas
 * Intellij IDEA
 * Formatos JSON
 * Lenguaje de programación JAVA
2. Metodología
Se implementó lo aprendido en las clases, utilizando una API publica.
Se consultó material de apoyo así como explicaciones dadas por Gemini con el objetivo de buscar alternativas a los bloqueos lógicos.
3. Funcionalidad
Al ser un proyecto sencillo, únicamente se ejecuta, se ingresa la opción de la moneda a convertir y se ingresa el monto a convertir. Finalmente, arroja el valor transformado, esto se repite hasta que el usuario coloque la opción de salir, dando fin al programa.
